# Operators

In order of precedence:

| Type | Operators |
| ---- | --------- |
| Unary | `!` `$` |
| Multiplicative | `*` `/` `%` |
| Additive | `+` `-` |
| Shift | `<<` `>>` |
| Relational | `<` `>` `<=` `>=` `<=>`|
| Equality | `==` `!=` `!~` |
| Bitwise (ordered) | `&` `^` `\|` |
| Logical (ordered) | `&&` `\|\|` |
| Ternary | `? :` |
| Assignment | `=` `+=` `-=` `*=` `/=` `%=` `<<=` `>>=` `&=` `^=` `\|=` |
